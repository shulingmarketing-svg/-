# 芓綾的行銷工作站

張書玲的個人品牌專案資料夾，包含 AI 知識分身設定、社群文案模板、行銷工具與品牌素材。

## 資料夾結構

```
/
├── ai-shuling-persona.md   # AI 芓綾人設與 System Prompt
├── social-content/         # 社群平台文案模板與發文規劃
├── tools/                  # HTML 小工具、數據整理腳本
├── brand/                  # 個人品牌素材、網站內容
└── news-digest/            # 每日/每週新聞重點整理
```

## 快速使用 AI 芓綾

1. 開啟 `ai-shuling-persona.md`
2. 複製裡面的 System Prompt
3. 貼入 Claude 專案設定或任何 AI 工具的系統提示欄位
4. 開始以芓綾身份工作
