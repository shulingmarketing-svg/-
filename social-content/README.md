# 社群文案資料夾

## 使用方式
把以下提示詞貼入 AI 芓綾專案，快速產出各平台文案。

## Facebook 文章提示詞模板

```
請以芓綾的口吻，針對「[主題]」撰寫一篇 Facebook 文章。
要求：
- 開頭吸引人，直接切入主題
- 語氣專業但親切，不生硬
- 內文 150-300 字
- 結尾加上一個行動呼籲
- 附上 3-5 個適合的 hashtag
```

## Instagram 貼文提示詞模板

```
請以芓綾的口吻，針對「[主題]」撰寫一篇 Instagram 貼文。
要求：
- 第一行要能抓住眼球（停止滑動）
- 內文簡短有力，100-150 字
- 條列式或分段，方便閱讀
- 5-8 個相關 hashtag
```

## 行銷觀點文章模板

```
請以芓綾的角度，針對「[行銷議題]」分享一個獨特觀點。
觀點方向：品牌力比促銷更重要、工具是加分不是核心、先做目標對齊再執行
字數：200-400 字
語氣：有觀點、有立場，但不說教
```

## 發文月曆規劃模板

```
請幫芓綾規劃「[月份]」的社群發文月曆。
平台：Facebook / Instagram
主題方向：行銷知識、AI 工具應用、品牌策略、個人觀點
頻率：每週 3-4 篇
格式：日期 | 主題 | 文章類型 | 重點關鍵字
```
